Welcome to the official GitHub organisation of the **University of Birmingham Computer Science Society (CSS)**!

This space is dedicated to building and maintaining
the software that powers the society.

## What You'll Find Here

* **Web Infrastructure:** The source code behind our website, [cssbham.com](https://cssbham.com) and committee wiki
* **Automation & Bots:** Specialised tools and integration bots that manage our Discord server, member verification, handle event logistics and interoperate with our Minecraft server
* **Open Source Sandboxes:** Repositories designed to help students learn Git workflows and submit their first Pull Requests

## How to Get Involved

You don’t need an invitation or an advanced degree in software engineering to contribute.
If you want to help out, we want your code!

1.  **Find an Issue:** Browse through our public repositories (like the website repo) and look for tags like `good first issue`
2.  **Fork and Code:** Experiment, add features or fix bugs
3.  **Submit a PR:** Our committee or repository maintainers will review your code, provide constructive feedback and then merge it
